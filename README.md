#Financial Transactions Analysis (SQL)

SQL project to analyze financial data and extract business insights like revenue trends, client performance, and growth.

## Features
Monthly & yearly revenue analysis
Top clients by revenue
Client ranking using window functions
Month-over-Month (MoM) growth
Running total (cumulative revenue)

 ## SQL Concepts
CTEs (WITH)
Window functions (LAG, RANK)
Aggregations (SUM, GROUP BY)
Date functions (DATE_TRUNC)

## Dataset
Contains transaction data with fields like:

date
amount
client_name

## Objective

Use SQL to analyze financial transactions and generate meaningful insights.
